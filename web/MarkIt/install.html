MarkIt 安装程序

使用说明：

  • 填写 "MarkIt 密钥" 后，点击 "更新链接"，然后将 "Mark It!" 链接拖到浏览器的书签栏(Bookmarks Toolbar)或右键添加到“收藏夹栏”(IE 8)，点击此书签，会在当前窗口显示或隐藏 MarkIt 对话框；
  • 请记录好 MarkIt 密钥，升级 "Mark It" 时，需要填入此密钥， 然后点击 "更新链接"，重做书签；
  • 适用 IE 8+、Chrome 3.0+、Firefox 3.5+、Safari 4+，不支持 Opera 10.60 以及 IE 6.0；
  • 与 Firefox 的 NoScript 扩展冲突，请禁用 NoScript，或者关闭 NoScript 配置中的 ABE 并允许此网站以及所访问网站的 JavaScript 脚本执行；
