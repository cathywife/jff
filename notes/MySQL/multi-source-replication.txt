Multi-source replication
========================


1. installation and configuration

    ./start-mysql-servers.sh

    # MySQL official APT repository:
    #   http://dev.mysql.com/downloads/repo/apt/

    # optionally
    apt install lsb-release wget whiptail
    dpkg -i mysql-apt-config_*_all.deb
    apt update


    filter options:

        #binlog-do-db=db1
        #binlog-ignore-db=test

        # http://dev.mysql.com/doc/refman/5.7/en/replication-rules.html
        # http://dev.mysql.com/doc/refman/5.7/en/change-replication-filter.html

        #replicate-do-db=db1
        #replicate-do-db=db2

        #replicate-ignore-db=test

        #replicate-do-table=db1.table1
        #replicate-do-table=db2.table2

        #replicate-ignore-table=db1.table1
        #replicate-ignore-table=db2.table2

        #replicate-wild-do-table=...
        #replicatewild-ignore-table=...
